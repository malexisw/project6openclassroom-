#Back End Piiquante

## Pour lancer le server
```
Renseigner l'URL de votre base de données dans le fichier config.js. Si vous voulez lancer le serveur sur un autre port que le 3000 changer alors le champ "port". 

 Ouvrer une invite de commande

Entrer la commande "npm install" pour installer les dépendances

Pour finir entrer la commande "npm run start" pour lancer le serveur sur le port choisit ou le port 3000.

```
